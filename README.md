# IPAYS Admin Prototype

这是一个可直接继续迭代的静态后台原型，风格参考了你提供的 IPAYS 管理后台截图。

## 当前已完成

- 后台整体框架：左侧菜单、顶部栏、页签条、内容区
- 一级菜单：`用户管理`
- 二级菜单：
  - `用户管理`
  - `用户认证审核管理`
- 用户管理列表页
- 用户详情页
  - `资料` tab：支持编辑并保存演示数据
  - `钱包` tab：显示余额、充值、提现、钱包明细
- 用户认证审核页
  - 状态：待审核、通过、拒绝
  - 审核弹窗：可手动指定通过后的钱包等级
  - 审核详情：包含证件照片展示区和人脸照片展示区，未上传时保留空态
- 菲律宾手机号格式校验：
  - `9xxxxxxxxx`
  - `09xxxxxxxxx`

## 文件说明

- `index.html`：页面入口
- `styles.css`：后台样式
- `app.js`：页面结构、路由、mock 数据、交互逻辑

## 使用方式

直接打开 `index.html` 即可预览。
